# QuestLearn Project Summary

## Final Project Name

`QuestLearn`

## Approved Tagline

`A Smart Interactive Learning System for Personalized Microlearning and Early Academic Support`

## Short Description

QuestLearn is a higher-education smart learning platform that combines guided microlearning, H5P/Lumi interactive content, automated quiz feedback, progress analytics, and advisor early alerts to create a more engaging and supportive learning experience.

## GitHub Repository Description

QuestLearn is a smart interactive learning platform for higher education featuring microlearning, H5P/Lumi content, quiz feedback, analytics, and advisor early alerts.

## Role-Based Feature Summary

### Student

- View courses, modules, and lessons
- Complete interactive lesson activities
- Attempt quizzes and receive feedback
- Track progress, weak topics, and recommendations
- Earn lightweight motivational rewards such as streaks, XP, and badges

### Instructor

- Create and manage courses, modules, and lessons
- Upload video and H5P/Lumi interactive content
- Build quizzes and question banks
- Review performance and engagement analytics

### Academic Advisor

- View assigned students and progress summaries
- Monitor advisor alerts and risk indicators
- Review support cues and follow up with students

### Admin

- Manage users and roles
- Approve instructor accounts
- Manage departments, announcements, and system oversight

## Portfolio Summary

QuestLearn is designed as both an academic systems project and a portfolio-ready product concept. It demonstrates role-based workflow design, interactive learning support, analytics-informed intervention, and practical platform planning for higher education.
