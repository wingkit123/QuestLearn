## Available Skills (Awesome Copilot Collection)

This project uses a curated combination of skills from the [Awesome GitHub Copilot](https://github.com/github/awesome-copilot) collection:

### Project Planning & Architecture

- **Acquire Codebase Knowledge** — Understand and document codebase structure
- **Architecture Blueprint Generator** — Create system architecture diagrams
- **Breakdown Epic / Feature / Plan** — Break down epics into tasks and create implementation plans

### Development & Documentation

- **Add Educational Comments** — Generate helpful code comments and documentation
- **Code Exemplars Blueprint Generator** — Generate code examples following best practices
- **Comment Code Generate A Tutorial** — Create tutorials from code

### Quality & Security

- **Agent OWASP Compliance** — Review code for security compliance
- **Agent Supply Chain** — Validate dependencies and supply chain security

### Research & Analysis

- **Automate This** — Identify automation opportunities in workflows
- **Autoresearch** — Research libraries, frameworks, and best practices
